# RSWEB – ASP.NET Core MVC Project


## Функционалности (Фаза 1)
- Управување со студенти, наставници и предмети (CRUD)
- Запишување студенти на предмети (Enrollment)
- Филтрирање и пребарување
- Прикачување и прикажување слики за студенти и наставници
- Entity Framework Core (Code First) + SQLite

## Технологии
- ASP.NET Core MVC
- Entity Framework Core
- SQLite
